# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Mini AliExpress project.

ADRs document significant architectural choices, their context, and consequences.

## Index

| ADR | Title | Status | Date |
|-----|-------|--------|------|
| [001](001-hexagonal-architecture.md) | Hexagonal Architecture for Backend | Accepted | 2025-01-01 |

## Creating a New ADR

1. Copy [template.md](template.md)
2. Name it `NNN-short-title.md` (increment the number)
3. Fill in all sections
4. Add an entry to the index table above
5. Submit via pull request
